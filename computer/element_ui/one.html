使用animation 后select-options宽度异常

异常图示：

出现原因：

它在运行动画时候，select组件的mounted就开始计算select框的宽度，导致了宽度没有正确计算。

解决方法：

 updated () {
    this.resizeSelect2()
    this.resizeSelect4()
  },
  methods: {
    // 修复下拉框选项宽度异常
    resizeSelect2 () {
      setTimeout(() => {
        this.$refs.selectInput2.resetInputWidth();
      }, 2000);
    },
 }
