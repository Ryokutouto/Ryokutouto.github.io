# 事件名称

# 鼠标事件

属性 描述 DOM
onclick (opens new window) 当用户点击某个对象时调用的事件句柄。 2
oncontextmenu (opens new window) 在用户点击鼠标右键打开上下文菜单时触发
ondblclick (opens new window) 当用户双击某个对象时调用的事件句柄。 2
onmousedown (opens new window) 鼠标按钮被按下。 2
onmouseenter (opens new window) 当鼠标指针移动到元素上时触发。 2
onmouseleave (opens new window) 当鼠标指针移出元素时触发 2
onmousemove (opens new window) 鼠标被移动。 2
onmouseover (opens new window) 鼠标移到某元素之上。 2
onmouseout (opens new window) 鼠标从某元素移开。 2
onmouseup (opens new window) 鼠标按键被松开。 2

# 键盘事件

属性 描述 DOM
onkeydown (opens new window) 某个键盘按键被按下。 2
onkeypress (opens new window) 某个键盘按键被按下并松开。 2
onkeyup (opens new window) 某个键盘按键被松开。 2

# 框架/对象（Frame/Object）事件

属性 描述 DOM
onabort (opens new window) 图像的加载被中断。 ( object) 2
onbeforeunload (opens new window) 该事件在即将离开页面（刷新或关闭）时触发 2
onerror (opens new window) 在加载文档或图像时发生错误。 ( object, body和 frameset)
onhashchange (opens new window) 该事件在当前 URL 的锚部分发生修改时触发。
onload (opens new window) 一张页面或一幅图像完成加载。 2
onpageshow (opens new window) 该事件在用户访问页面时触发
onpagehide (opens new window) 该事件在用户离开当前网页跳转到另外一个页面时触发
onresize (opens new window) 窗口或框架被重新调整大小。 2
onscroll (opens new window) 当文档被滚动时发生的事件。 2
onunload (opens new window) 用户退出页面。 ( body 和 frameset) 2

# 表单事件

属性 描述 DOM
onblur (opens new window) 元素失去焦点时触发 2
onchange (opens new window) 该事件在表单元素的内容改变时触发( input, keygen, select, 和 textarea) 2
onfocus (opens new window) 元素获取焦点时触发 2
onfocusin (opens new window) 元素即将获取焦点时触发 2
onfocusout (opens new window) 元素即将失去焦点时触发 2
oninput (opens new window) 元素获取用户输入时触发 3
onreset (opens new window) 表单重置时触发 2
onsearch (opens new window) 用户向搜索域输入文本时触发 ( input="search")
onselect (opens new window) 用户选取文本时触发 ( input和 textarea) 2
onsubmit (opens new window) 表单提交时触发 2

# 剪贴板事件

属性 描述 DOM
oncopy (opens new window) 该事件在用户拷贝元素内容时触发
oncut (opens new window) 该事件在用户剪切元素内容时触发
onpaste (opens new window) 该事件在用户粘贴元素内容时触发

# 打印事件

属性 描述 DOM
onafterprint (opens new window) 该事件在页面已经开始打印，或者打印窗口已经关闭时触发
onbeforeprint (opens new window) 该事件在页面即将开始打印时触发

# 拖动事件

事件 描述 DOM
ondrag (opens new window) 该事件在元素正在拖动时触发
ondragend (opens new window) 该事件在用户完成元素的拖动时触发
ondragenter (opens new window) 该事件在拖动的元素进入放置目标时触发
ondragleave (opens new window) 该事件在拖动元素离开放置目标时触发
ondragover (opens new window) 该事件在拖动元素在放置目标上时触发
ondragstart (opens new window) 该事件在用户开始拖动元素时触发
ondrop (opens new window) 该事件在拖动元素放置在目标区域时触发

# 多媒体（Media）事件

事件 描述 DOM
onabort (opens new window) 事件在视频/音频（audio/video）终止加载时触发。
oncanplay (opens new window) 事件在用户可以开始播放视频/音频（audio/video）时触发。
oncanplaythrough (opens new window) 事件在视频/音频（audio/video）可以正常播放且无需停顿和缓冲时触发。
ondurationchange (opens new window) 事件在视频/音频（audio/video）的时长发生变化时触发。
onemptied 当期播放列表为空时触发
onended (opens new window) 事件在视频/音频（audio/video）播放结束时触发。
onerror (opens new window) 事件在视频/音频（audio/video）数据加载期间发生错误时触发。
onloadeddata (opens new window) 事件在浏览器加载视频/音频（audio/video）当前帧时触发触发。
onloadedmetadata (opens new window) 事件在指定视频/音频（audio/video）的元数据加载后触发。
onloadstart (opens new window) 事件在浏览器开始寻找指定视频/音频（audio/video）触发。
onpause (opens new window) 事件在视频/音频（audio/video）暂停时触发。
onplay (opens new window) 事件在视频/音频（audio/video）开始播放时触发。
onplaying (opens new window) 事件在视频/音频（audio/video）暂停或者在缓冲后准备重新开始播放时触发。
onprogress (opens new window) 事件在浏览器下载指定的视频/音频（audio/video）时触发。
onratechange (opens new window) 事件在视频/音频（audio/video）的播放速度发送改变时触发。
onseeked (opens new window) 事件在用户重新定位视频/音频（audio/video）的播放位置后触发。
onseeking (opens new window) 事件在用户开始重新定位视频/音频（audio/video）时触发。
onstalled (opens new window) 事件在浏览器获取媒体数据，但媒体数据不可用时触发。
onsuspend (opens new window) 事件在浏览器读取媒体数据中止时触发。
ontimeupdate (opens new window) 事件在当前的播放位置发送改变时触发。
onvolumechange (opens new window) 事件在音量发生改变时触发。
onwaiting (opens new window) 事件在视频由于要播放下一帧而需要缓冲时触发。

# 动画事件

事件 描述 DOM
animationend (opens new window) 该事件在 CSS 动画结束播放时触发
animationiteration (opens new window) 该事件在 CSS 动画重复播放时触发
animationstart (opens new window) 该事件在 CSS 动画开始播放时触发

# 过渡事件

事件 描述 DOM
transitionend (opens new window) 该事件在 CSS 完成过渡后触发。

# 其他事件

事件 描述 DOM
onmessage 该事件通过或者从对象(WebSocket, Web Worker, Event Source 或者子 frame 或父窗口)接收到消息时触发
onmousewheel 已废弃。 使用 onwheel (opens new window) 事件替代
ononline (opens new window) 该事件在浏览器开始在线工作时触发。
onoffline (opens new window) 该事件在浏览器开始离线工作时触发。
onpopstate 该事件在窗口的浏览历史（history 对象）发生改变时触发。
onshow (opens new window) 该事件当 menu 元素在上下文菜单显示时触发
onstorage 该事件在 Web Storage(HTML 5 Web 存储)更新时触发
ontoggle (opens new window) 该事件在用户打开或关闭 元素时触发
onwheel (opens new window) 该事件在鼠标滚轮在元素上下滚动时触发

# 常量

静态变量 描述 DOM
CAPTURING-PHASE 当前事件阶段为捕获阶段(1) 1
AT-TARGET 当前事件是目标阶段, 在评估目标事件(1) 2
BUBBLING-PHASE 当前的事件为冒泡阶段 (3) 3

# 属性

属性 描述 DOM
bubbles (opens new window) 返回布尔值，指示事件是否是起泡事件类型。 2
cancelable (opens new window) 返回布尔值，指示事件是否可拥可取消的默认动作。 2
currentTarget (opens new window) 返回其事件监听器触发该事件的元素。 2
eventPhase 返回事件传播的当前阶段。 2
target (opens new window) 返回触发此事件的元素（事件的目标节点）。 2
timeStamp (opens new window) 返回事件生成的日期和时间。 2
type (opens new window) 返回当前 Event 对象表示的事件的名称。 2

# 方法

方法 描述 DOM
initEvent() 初始化新创建的 Event 对象的属性。 2
preventDefault() 通知浏览器不要执行与事件关联的默认动作。 2
stopPropagation() 不再派发事件。 2

# 鼠标/键盘事件对象

# 属性

属性 描述 DOM
altKey (opens new window) 返回当事件被触发时，"ALT" 是否被按下。 2
button (opens new window) 返回当事件被触发时，哪个鼠标按钮被点击。 2
clientX (opens new window) 返回当事件被触发时，鼠标指针的水平坐标。 2
clientY (opens new window) 返回当事件被触发时，鼠标指针的垂直坐标。 2
ctrlKey (opens new window) 返回当事件被触发时，"CTRL" 键是否被按下。 2
Location (opens new window) 返回按键在设备上的位置 3
charCode (opens new window) 返回onkeypress事件触发键值的字母代码。 2
key (opens new window) 在按下按键时返回按键的标识符。 3
keyCode (opens new window) 返回onkeypress事件触发的键的值的字符代码，或者 onkeydown 或 onkeyup 事件的键的代码。 2
which (opens new window) 返回onkeypress事件触发的键的值的字符代码，或者 onkeydown 或 onkeyup 事件的键的代码。 2
metaKey (opens new window) 返回当事件被触发时，"meta" 键是否被按下。 2
relatedTarget (opens new window) 返回与事件的目标节点相关的节点。 2
screenX (opens new window) 返回当某个事件被触发时，鼠标指针的水平坐标。 2
screenY (opens new window) 返回当某个事件被触发时，鼠标指针的垂直坐标。 2
shiftKey (opens new window) 返回当事件被触发时，"SHIFT" 键是否被按下。 2
